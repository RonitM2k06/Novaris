# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-24

### Added
- **Phase 0-3:** Core Economic Knowledge Graph, Simulation Engine, Explainability Engine, and Causal Calibration Engine.
- **Phase 4:** Real Data Integration (FRED/World Bank/OECD APIs).
- **Phase 5:** Historical Replay Engine and validation against 5 canonical crises.
- **Phase 6:** Ontology Expansion (Financial, Policy, Behavioral systems).
- **Phase 7:** Nonlinear Dynamics Engine (Saturation/Substitution effects using `tanh`, Policy Dampening).
- **Phase 8:** Sticky Price & Adaptive Agent Engine (Asymmetric response functions for deflation, wage stickiness, adaptive expectations via EMA).
- **Phase 9:** Next.js Command Center (Ultra Dark UI, Tabbed Workstation).
